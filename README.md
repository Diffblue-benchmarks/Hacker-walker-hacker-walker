### 使用说明 
```
1.使用命令拉取代码：
    git clone https://gitee.com/walker-cloud/Hacker-walker.git 
2.创建数据库：jinnian(堇年)， 字符集：utf8;（注意：只需要你创建数据库即可）
3.最好使用IDEA导入该项目
4.修改配置
    A.打开application.properties，配置数据库连接
          datasource:
              url: 你的数据库地址
              username: 你的数据库用户名
              password: 你的数据库密码
5.运行项目(后台管理模块)
    a.直接运行hacker.channel.manage目录下的App
    b.浏览器访问：http://127.0.0.1:8888
```
